## Brand Synopsis

**Don't Die Alone** is a brutal, evidence-based read on a man's dating profile. Upload your screenshots → get a platform-specific score, per-photo verdict, bio + prompt breakdown, and a top-fixes list ranked by estimated match-rate lift. Built by Zack Ballantine under the Banana Bytes umbrella. Web app at `dontdiealone.app/scan`. Free during beta.

The brand voice is **brutal-but-kind**: confident, witty, evidence-backed, never desperate, never cruel. The product references published research from Hinge, Bumble, Photofeeler, and peer-reviewed studies. The verdict text is generated by Claude against a structured rubric — not freeform chat.

The audience is men 25–45 who use dating apps and aren't getting the matches they expect.

## Design Decisions

### Direction A · Locker Room (`direction-locker-room.html`)
- **Mood**: Friday-night-football scoreboard energy. Numbers as truth.
- **Fonts**: Barlow Condensed (display, weight 800/900) + Barlow (body) — both Google Fonts
- **Palette**:
  - `#0A0A0A` Off-court black (bg)
  - `#FAFAFA` Floodlight white (text)
  - `#FBBF24` Scoreboard amber (CTA / numbers)
  - `#DC2626` Ref-stripe red (warnings / Fuck-Boy Detector)
- **Layout**: Scoreboard hero anchor + LED-styled stat panels + ref-stripe section dividers + bench-card features
- **Signature element**: A1 scoreboard hero rendering the 0.6% / 10.5% gut-punch as a literal Friday-night LED board. Plus LED-glow CSS effect on every numeric stat.
- **Logos used (Logo Search)**: none — direction is sport-aesthetic-only, no brand logos required
- **Trends used (TRENDS_QUICK_REFERENCE)**: bold-block typography, scrolling-marquee-adjacent (none used here, kept to the scoreboard register), colored borders + offset shadows on cards

### Direction B · Editorial (`direction-editorial.html`)
- **Mood**: A New Yorker piece on why dating is broken.
- **Fonts**: Libre Bodoni (display, italic-leaning) + Public Sans (body)
- **Palette**:
  - `#FAFAFA` Newsprint cream (bg)
  - `#09090B` Ink black (text)
  - `#EC4899` Brand pink (accent — kept from product palette)
  - `#3F3F46` Graphite (secondary text)
- **Layout**: Cover treatment with kicker ("Issue 001 · Vol. 1 · Winter 2026 · The Dating Issue") + drop-cap intro + 2-column prose + pull-quote section break + Roman-numeral feature lists
- **Signature element**: Issue 001 cover treatment + drop cap + 5xl/6xl pull quote in italic Bodoni
- **Logos used**: none
- **Trends used**: editorial-magazine layout, hairline drop rules, oversized italic display type

### Direction C · Arcade (`direction-arcade.html`)
- **Mood**: Pizza-shop arcade cabinet rebrand. Y2K chrome, dive-bar neon, character-select energy.
- **Fonts**: Anton (display) + Epilogue (body) + Press Start 2P (game UI labels)
- **Palette**:
  - `#0A0A0A` CRT black (bg)
  - `#EC4899` Hot pink neon (primary)
  - `#22D3EE` Electric cyan (secondary)
  - `#FACC15` Lemon yellow (highlight / score)
- **Layout**: Top + bottom marquee tickers, neon arcade-cabinet hero, fighting-game lifebar stats, character-select feature grid, pixel-card offset shadows
- **Signature element**: HP/HP lifebar showing P1 (You) at 6% vs P2 (Women) at 100% on the Tinder match-rate stat. Plus the C3 banana-suit pixel sprite as a hidden founder Easter egg with caption "★ Hidden character unlocked ★".
- **Logos used**: none
- **Trends used**: scrolling marquee tickers, neon glow text-shadow stack, CRT scanline overlay, chunky pixel-art borders + offset hard shadows

### Direction D · Quiet (`direction-quiet.html`)
- **Mood**: Calm.app for dating sanity. The opposite of every other direction.
- **Fonts**: Lora (serif display) + Raleway (body sans)
- **Palette**:
  - `#FAFAF7` Warm off-white (bg)
  - `#0A0A0A` Deep ink (text)
  - `#A3B18A` Soft sage (accent)
  - `#C9A66B` Muted clay (secondary accent)
- **Layout**: Single column, generous margins, near-blank opening hero (84vh of cream paper with single italic line), single-line section dividers, footnote-style citations
- **Signature element**: Blank-page first impression. The opening section is intentionally near-empty — just the phrase "a kindness, plainly." in italic Lora serif at 5xl, centered, with two faint sage rule lines. Stats reveal as small inline numbers in the body prose with footnote-style ¹²³ markers.
- **Logos used**: none
- **Trends used**: extreme negative space, single-column reading flow, hand-drawn-style horizontal rules, footnote citations

## Production Notes

To build this into a real production site, use Claude Code (Opus, high effort).

If shipping just **one** of these directions, my recommendation is **A · Locker Room** as the public-facing default (the scoreboard is the strongest single visual hook), with **D · Quiet** as a `/quiet` alternate route for mood-sensitive visitors who land via the founder's mental-health work or Banana Cares. Editorial (B) and Arcade (C) are stronger as social-share / campaign-specific landing pages than as the daily-driver site.

Production stack recommendation: Next.js 16 on Vercel Edge (matches the existing app stack per BRIEF technical notes). The 4 mockups are vanilla HTML right now but the structures port to Next.js cleanly — one component per section, the `reveal` IntersectionObserver becomes a `useInView` hook or Framer Motion variant, and the Tailwind CDN becomes a properly configured `tailwind.config.ts` with the 4 palettes as theme variants.

If Daddy wants to A/B test directions in the wild, all 4 are independently deployable as static HTML files behind a feature flag — no rebuild needed.
